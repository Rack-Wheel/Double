## 二


一动不动地站在太阳底下的滋味，但凡是品尝过的人，想必是不愿再次回味吧。今天是军训第第七天——转折点。最后一天是结训破冰晚会，每个方队需要出一个节目，并且学校自己组织了两个节目：集体舞、合唱。今天是为这学校组织的节目报名的时间，经过选拔，选上的人可以免去之后下午的军训，去参与排练。

<span style='color:#EE6600'>“老哥，这个可好啊——我可不想继续晒下去了啊——都要晒黑了。”</span>

<span style='color:#E60000'>“这个必须得报一个啊！”</span>

<span style='color:#EE6600'>“那咱们报舞蹈——”</span>

<span style='color:#E60000'>“合唱！拒绝跳舞！”</span>

书要简言，参加了下午的清唱式选拔，轻易地进入了合唱队。

第二天，经历了一上午的站立、正步，中午我倒在床上虚脱。暗暗地庆幸自己选择了合唱团——外面太阳正烈。下午的排练不需要着军训服，我便冲了个澡，换了舒适的私服。倒在床上小憩三十分钟，边走去了集合地点——活动中心门前。到那边之后，一个打扮的很精致的学长把我们带到了三楼的多功能厅。

来在了多功能厅之后，那个学长开始给我们排队形了。我环顾四周：这多功能厅是一个数百平米的房间，纵向的一头有小型的舞台、投影设备、两侧音响。正对舞台的右侧有一个老旧的调音台。那个学长给我们排好了队形——最前面是两个学长和两个学姐领唱，每个人拿一个话筒，其后面全是大一新生。我则站在中间排中间列。

那个带我们来的学长给我们排好队形之后，走到了调音台那里摆弄了起来。他把音响电源接通了以后，将音频线插入自己的电脑。这一瞬间，音响发出了“吱——嘭”的巨响。有几个同学发出了小声的惊呼。

<span style='color:#E60000'>“音响带电插拔可是大忌讳啊——学校也不找个懂的人来搞。”</span>

那个学长按下了播放键，开始播放背景音乐。

<span style='color:#E60000'>“你觉着这音量怎么样？”</span>

<span style='color:#EE6600'>“我感觉正好啊——”</span>

<span style='color:#E60000'>“一会儿唱起来，你就听不见了。”</span>

果然不出他所料，唱到一半的时候，唱的歌和配乐已经完全不在一个拍子上了——大家基本都听不到配乐的声音。单纯合唱的声音就可以把背景音乐盖过去，但最致命的是那四个领唱的人的话筒——四个人的声音响度不同，而且还有两个人的巨响。

“你们这节奏太差了啊！”

那个学长开始埋怨我们。可以从大家的脸色上看出，每个人心里都有一些不爽——你的声音太小了啊。但他们都不说——

<span style='color:#E60000'>“学长，麻烦你声音大一点，我们听不清楚！”</span>

他喊了出来。每个人都看向他——我。那个学长盯了我一会儿，跑到电脑旁边，捣鼓了一下，回过身来，拿着话筒，背对着音响冲着我们说：

“已经最大了！”

进而持续而尖锐的声音从音响中传出来。那个学长开始有点慌了。大家纷纷捂住了耳朵。

他在心里对我讲：<span style='color:#E60000'>“再业余也不能这么菜吧。”</span>

于阳小跑过去，把话筒从音响哪里指向别处，声音继而停止了。他又转到调音台那里，抬头看向四个领唱。

<span style='color:#E60000'>“左边的学姐你唱一句。”</span>

那个学姐楞了一下。

<span style='color:#E60000'>“麻烦唱一句。”</span>

她这才缓过神，唱了几句。

“你在干什么？”那个学长问于阳。

<span style='color:#E60000'>“在帮你调啊——另一个学姐也唱一句。”</span>

另一个学姐开始了她的表演，这时，门口又进来一个女生。管事的学长走过去跟那个女生说了些什么——估计她是我的学姐吧。

继而四个人都唱过了：

<span style='color:#E60000'>“麻烦四位都唱一下，就从XXXX那里开始，来，一、二、三、四。”</span>

四位同时唱，于阳把推子推了推，这样他们四个的声音听起来响度相同了。

他在心里对我说：<span style='color:#E60000'>“懒得调音色了。”</span>

<span style='color:#E60000'>“学长，能不能帮我放一下背景音乐啊。”</span>

他考虑到这是私人电脑，不适合自己去操作。

<span style='color:#00da00'>		“我来给你弄。”</span>

那个学姐走了过来，用那台电脑点了播放键。

于阳把音频线对应的推子上面的增益旋钮拧了一下，声音便大了不少。他又伴着伴奏调了调四位领唱的总体响度。

继而下午的排练顺利地进行着。

----------------------------------

时间渐渐到了晚上，身体他也让给了我。我在学校里散步，享受已有一丝秋意的傍晚。走到操场边的辅路，看到了很多社团支着帐篷发报名表——到了社团招新的时候了。眼睛偶然撇到了旁边有个帐篷边摆着海报，写着“电声乐团”。

<span style='color:#EE6600'>“老哥，这个正合你意啊！”</span>

<span style='color:#E60000'>“哦哟！”</span>

<span style='color:#EE6600'>“来来来，身体给你——上午军训的时候都让你扛着，也挺辛苦的，算我给犒劳你吧。”</span>

他接过身体之后，走到帐篷前，拿了一张报名表，看了看内容，准备问问面试的时间：

<span style='color:#E60000'>“请问一下，这个面试——欸？”</span>

<span style='color:#00da00'>		“嗯？啊，你好啊！”</span>

<span style='color:#00da00'>		“你是下午那个，调音的女孩子吧！”</span>

<span style='color:#E60000'>“呃——是的。”</span>

女孩子——噗。

<span style='color:#E60000'>“我想报名电声乐团——面试是——”</span>

<span style='color:#00da00'>		“乐器是？”</span>

<span style='color:#E60000'>“电吉他。”</span>

<span style='color:#00da00'>		“现在跟我过来。”</span>

<span style='color:#E60000'>“哈？”</span>

学姐跟身后的人打了声招呼，便拉着于阳到了电声乐团的排练室。

<span style='color:#E60000'>“学姐你是——”</span>

<span style='color:#00da00'>		“啊——我是电声乐团的团长来着，弹BASS的。“</span>

<span style='color:#E60000'>“下午的合唱是电声乐团负责吗？”</span>

<span style='color:#00da00'>		“我还兼任校学生会文艺部副部长来着。“</span>

<span style='color:#00da00'>		“你是哪个院的？“</span>

<span style='color:#E60000'>“通信学院。”</span>

<span style='color:#00da00'>		“那我还兼任咱院学生会文艺部副部长来着。“</span>

这也太现充了吧！

于阳对我讲：<span style='color:#E60000'>“真可怕。”</span>

<span style='color:#00da00'>		“闲话少说，咱们过个形式——走个流程吧。“</span>

已经明说是过个形式吗！

于阳借用了摆在一旁的电吉他，插上电，弹了一段FUNK。

<span style='color:#00da00'>		“好了，录用。“</span>

好快！

<span style='color:#E60000'>“这也太草率了吧！”</span>

<span style='color:#00da00'>		“效果器啥的也没问题吧。“</span>

<span style='color:#E60000'>“当然——咱也玩过混音。”</span>

<span style='color:#00da00'>		“那还有什么可说的吗？录用了。“</span>

<span style='color:#00da00'>		“木吉他也会吗？“</span>

<span style='color:#E60000'>“没电吉他熟练——简单弹唱倒是没问题。”</span>

<span style='color:#00da00'>		“作为团员的第一个任务：明天傍晚来帐篷帮忙迎新！“</span>

<span style='color:#E60000'>“没问题。”</span>

这是我和于阳并没有考虑她为何会问我会不会木吉他。这是我和于阳今天掉的第一个坑。

于阳感觉非常尴尬，不知该继续说些什么。他稍稍缓和了一下心情，开口了：

<span style='color:#E60000'>“学姐身兼多职，真的忙得过来吗？”</span>

<span style='color:#00da00'>		“其实吧，三者不怎么冲突，活动都错开了。“</span>

<span style='color:#00da00'>		“不过这个破冰晚会可是够累的。学校的节目和方队出的节目都得文艺部去审——正部长又不是很懂。“</span>

那个学长原来是正部长吗？

<span style='color:#00da00'>		“你看今天下午的排练，我不在，不就出问题了吗——刚开始我在集体舞那边来着。“</span>

<span style='color:#00da00'>		“等一下——“</span>

<span style='color:#00da00'>		“你是特别想去上台唱吗？“</span>

<span style='color:#E60000'>“呃——显然不是吧，我只是不想在太阳底下站着——”</span>

<span style='color:#00da00'>		“我去年这个时候也是这么想的——“</span>

<span style='color:#00da00'>	“之后的合唱不要参加了，调音能不能都拜托你了啊！”</span>

<span style='color:#00da00'>	“我其他活实在太多了。”</span>

<span style='color:#E60000'>“这到没问题。”</span>

他对我讲：<span style='color:#E60000'>“这简直美滋滋吧！每天下午去那里坐着就可以了！”</span>

<span style='color:#00da00'>	“你可要言而有信哦！”</span>

<span style='color:#E60000'>“放心，一言既出，驷马难追。”</span>

这便是我和于阳掉的第二个坑！
